如何使用

安装mkdocs

安装Python后输入，以安装与本文档相同的mkdocs主题 

pip install mkdocs-material

并且可在此处查看其使用方法
现在只需要push到main分支中就可以自动生成网页到gh-pages分支中，不再需要手动生成了

编辑文本

添加新的任务请按月份和任务类型分解

    ##5月

        * [x] 子任务2
